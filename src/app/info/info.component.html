Dicionario de dados

  • Order ID - Um ID de pedido é o sistema numérico que a Amazon usa exclusivamente para rastrear pedidos. Cada pedido recebe seu próprio ID de pedido que não será duplicado. Este número pode ser útil para o vendedor ao tentar descobrir certos detalhes sobre um pedido, como data de envio ou status.
  • Produto - O produto que foi vendido.
  • Quantidade - A quantidade total de itens encomendados no pedido inicial (sem nenhuma alteração).
  • Preço unitário - The price of each products.
  • Data de Compra - Esta é a data em que o cliente solicita o envio do pedido.
  • Endereço de Envio - O pedido de compra é preparado pelo comprador, muitas vezes por meio de um departamento de compras. A ordem de compra, ou PO, geralmente inclui um número de PO, que é útil para combinar remessas com compras; uma data de envio; Endereço de Cobrança; Endereço para envio

Objetivo

Analisar os dados de vendas da Amazon para extrair informações relevantes sobre:

  • Cidades em que gerão mais lucratividade
  • Produtos com maior fluxo de saída
  • Produtos que geram mais receita
  • Construção da série temporal para analisar os dias, semas e meses com mais vendas e tendências
  • Horarios do dia com maior fluxo de pedidos
